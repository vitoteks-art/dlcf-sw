# DLCF Platform Quick Guide (1 Page)

## Public Visitor (No Login)
1. Open the website.
2. Use the top menu:
   - About Us, What We Believe
   - Media and Publications
   - States
3. Click any Media or Publication for details.
4. On a State page, use State Media and State Publications.

## Logged-in User
1. Click Portal and log in.
2. Fill Biodata once.
3. Register for:
   - State Congress
   - Zonal Congress
   - Retreat
   - STMC
4. Submit Attendance and GCK if assigned.

## Admin
1. Log in and open Admin.
2. Manage:
   - Users, Locations, Organization, Roles
   - State Home and State Updates
   - Congress dates
   - Media and Publications
3. Publish items to make them visible.

## Help
If something is missing, check:
- Status = Published
- Correct dates
- Correct state selected
